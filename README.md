# FlyingSquid
ephemeral HTTP communication, self-handled encyption protocol

## USAGE
in Resources/Scripts/ :
- modify encode.sh to configure your own encryption method
- modify decode.sh to configure the decryption method your friend tells you
- startserver.sh can be modified to do some extras like run the server in a virtual environement

## EXECUTION
```bash
./FlyingSquid.sh
```
